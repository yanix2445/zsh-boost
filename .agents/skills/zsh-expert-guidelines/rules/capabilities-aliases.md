# Common Aliases

**Impact: LOW**

Utiliser les alias optimisés pour la productivité.

## Navigation & Fichiers
- `ls` → `eza` (si installé) : Listage coloré avec icônes.
- `cat` → `bat` (si installé) : Affichage avec coloration syntaxique.
- `grep` → `rg` (ripgrep) : Recherche ultra-rapide.
- `z` / `zz` : Saut de dossier intelligent (zoxide).
- `..` : Remonter d'un dossier.

## Git (Oh My Zsh)
- `gst` : `git status`
- `gco` : `git checkout`
- `gp` : `git push`
- `gl` : `git pull`
- `glog` : Log graphique (alias custom).

## Dev
- `d` : `docker`
- `dc` : `docker-compose`
- `ni` : `npm install`
- `nrun` : `npm run`
- `p` : `pnpm`
- `b` : `bun`
